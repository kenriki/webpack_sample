
# webpack の使い方

# 導入準備
1. node.jsパッケージ ←公式からダウンロードしてインストール
2. [1]を環境変数に設定する
3. devフォルダ作成
4. cd dev
5. npm install ←node_modules/ 生成
6. npm run build ←dist配下にファイル生成
